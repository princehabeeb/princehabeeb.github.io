My UX skills

USER RESEARCH

User research is essential in identifying challenges and opportunities to improve the user experience. We are conducting the research by doing user interviews, surveys, card sorting, and usability testing.

USER EMPATHY

User empathy is done is 4 steps: Define, Ideate, Prototype and Test. By creating a Persona, an user journey, a storyboard, and clearly understanding what the user needs, we can create a product that will cater to user interest and needs.

QUALITATIVE INTERVIEWS

Based on a created interview script including a description of why we are conducting the interview, who we are and getting an agreement to record and share the answers, we ask questions that are impactful to our project. We make sure that the interviews are neutral, with open ended questions.
